1.数据排序

有两种排序方法，就是series和dataframe
首先

    import pandas as pd
    path = './daily_price_btc_cny.csv'
    df = pd.read_csv(path)

3.index操作

    import numpy as np
    df =pd.DataFrame(
        np.arange(12).reshape(3,4),
        columns=['A','B','C','D']
    )
    df
    产生这样一个3行4列的数据表
        A    B    C    D
    0    0    1    2    3
    1    4    5    6    7
    2    8    9    10    11

3.1删除某一列

    df.drop('A',axis=1)
    #axis=1表示列,或者axis=column
    结果：
        B    C    D
    0    1    2    3
    1    5    6    7
    2    9    10    11

3.2删除某一行

    df.drop('A',axis=1)
    #axis=1表示列,或者axis=column
    结果：
    A    B    C    D
0    0    1    2    3
2    8    9    10    11

3.3按axis=0/index执行mean聚合操作–遍历行，找到列的均值

    按axis=0/index执行mean聚合操作--遍历行，找到列的均值
    结果：
    A    4.0
    B    5.0
    C    6.0
    D    7.0
    dtype: float64

3.4按axis=1/column执行mean聚合操作–遍历列，找到行的均值

    df.mean(axis=1)
    结果：
    0    1.5
    1    5.5
    2    9.5
    dtype: float64
