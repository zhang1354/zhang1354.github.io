五、数据分析第五课-pandas查询数据-loc、iloc

1.查询数据

##　loc查询通用公式:df.loc[‘条件’,’列名’]
###　1.0读取数据

    import pandas as pd
    df = pd.read_csv("./daily_price_btc_cny.csv")
    df.head()
    结果:
    date    open    high    low    close    volume
0    2013/9/1    806.37    815.0    803.20    811.42    84.2276
1    2013/9/2    810.60    816.7    801.00    811.00    317.1206
2    2013/9/3    809.70    812.0    797.90    801.37    304.0173
3    2013/9/4    801.00    802.2    797.00    798.20    308.5300
4    2013/9/5    800.00    802.0    663.27    752.30    1016.1104

1.1数据初步替换

替换1-3行的字符，将/替换为-

    #将日期中的/替换为- 1:3表示只替换第一行到第三行
    df.loc[1:3,'date'] = df['date'].str.replace("/","-")
    df.head()
    结果:
    date    open    high    low    close    volume
0    2013/9/1    806.37    815.0    803.20    811.42    84.2276
1    2013-9-2    810.60    816.7    801.00    811.00    317.1206
2    2013-9-3    809.70    812.0    797.90    801.37    304.0173
3    2013-9-4    801.00    802.2    797.00    798.20    308.5300
4    2013/9/5    800.00    802.0    663.27    752.30    1016.1104
    替换所有行
    # ：表示替换所有行
    df.loc[:,'date'] = df['date'].str.replace("/","-")
    df.head()

1.2查询某行多列的数据

查询2013-9-1行的open、high列的两个数据

    df.set_index('date',inplace=True)
    df.loc['2013-9-1',['open','high']]
    结果:
    open    806.37
    high    815.00
    Name: 2013-9-1, dtype: float64

1.3查询多行多列的数据

查询2013-9-1’,’2013-9-2两行和open、high四个数据

    df.loc[['2013-9-1','2013-9-2'],['open','high']]
    结果:
                open    high
    date        
    2013-9-1    806.37    815.0
    2013-9-2    810.60    816.7

1.4使用数据区间进行查询

查询2013-9-1到2013-9-2的open到high的数据

    df.loc['2013-9-1':'2013-9-2','open':'close']
                open    high    low    close
    date                
    2013-9-1    806.37    815.0    803.2    811.42
    2013-9-2    810.60    816.7    801.0    811.00

1.5使用条件表达式查询（>、<之类的）

查询所有行中open<10000的数据，“df[‘open’]<10000”是查询条件，:表示选择所有列

    df.loc[df['open']<10000,:]
    #两个条件用&
    df.loc[(df['open']<10000)&(df['high']>9000),:]

2.新增数据列

2.1直接赋值方法，新增数据列

    #新增一个最高价格与最低价格差的列“jiacha”=最高价-最低价，赋值给所有行
    df.loc[:,'jiacha']=df['high']-df['low']
    df.head()

2.2 df.apply方法

apply方法就是可以创建一个条件函数，将条件封装在函数里，然后创建符合条件的列
如下：先创建一个type函数，df.apply(type,axis=1)表示创建一个

    #axis=1表示添加列
    def type(x):
        if x['open']>9000:
            return '高'
        if x['open']<8000:
            return '低'
        return '正常'
    df.loc[:,'type'] = df.apply(type,axis=1)
    df.head()

2.3按条件分组分别赋值

    #先创建空列
    df['price_cha']=''
    df.loc[df['high']-df['low']>1000,'price_cha'] = '价格差别大'
    df.loc[df['high']-df['low']<100,'price_cha'] = '价格差别小'
    df.head()
