三、数据分析第三课-python基本数据结构

1数据结构和序列

1.1元组

元组是⼀个固定⻓度，不可改变的Python序列对象。创建元组的最简单⽅式，是⽤逗号分隔⼀列值：

1.1.1创建

    #tuple--元组，创建非常简单，直接用逗号就可以了,或者是用括号+逗号
    #创建tuple
    tup1 = (4,5,6)
    type(tup1)
    tup2 = 1,2,3
    type(tup2)

1.1.2将列表转换为tuple

    #将列表转换为tuple
    b = [1,2,3]
    a = tuple(a)
    type(a)

1.1.3tuple没法直接修改，没有append、delete，只要创建了就不可改变。

但是tuple中如果包含可以改变的数组比如list，可以修改list中的内容

    c = ([1,2,3],5)
    c[0].append(4)
    c
    结果：
    ([1, 2, 3, 4], 5)

1.1.4元组可以使用“+”直接将两个元组加起来

    d = 1,2,3
    e = 4,5,6
    d+e
    结果：
    (1, 2, 3, 4, 5, 6)

1.1.5元组拆分：

    #元组拆分
    f = 1,2,3
    a,b,c=f
    a
    结果：
    1

1.2列表

除去一些基本功能，将不知道的list功能记录一下

1.2.1添加和删除元素

apend、insert、pop、remove

    #append在末尾添加元素，在某地插入元素用insert
    list_1 = [1,2,3,5]
    list_1.insert(3,4)
    list_1
    结果：
    [1, 2, 3, 4, 5]

    #删除元素--pop
    #insert的逆运算是pop，删除第几个元素
    list_1.pop(0)
    list_1
    结果
    [2,3,4,5]

    #删除元素--remove
    #remove删除某个值，默认移除找到的第一个值
    list_2 = [1,2,3,4,5,6,"aaa"]
    list_2.remove("aaa")
    list_2
    结果：
    [1,2,3,4,5,6]

1.2.2 extend函数

如果已经定义了一个列表，使用append可以添加元素，使用extend可以追加多个元素，可以追加另一个列表，

    everything    =    [] 
    for    chunk    in    list_of_lists:                
        everything.extend(chunk)
    #extend比下述方法更快
    everything    =    [] 
    for    chunk    in    list_of_lists:                
        everything    =    everything    +    chunk

1.2.3排序

sort()函数

    #sort函数
    a = [1,2,5,4,3,5,8,1,]
    a.sort()
    a
    结果：
    [1, 1, 2, 3, 4, 5, 5, 8]

1.2.4切片

正向切片的结果是左闭右开的

    #切片
    b = [1,2,3,4,5,6,7,8,9]
    b[1:5]
    结果：
    [2, 3, 4, 5]

反向切片的结果也是左开右闭的，因为负数切片也是从0(就是从右边）开始，然后从右往左数，第一个是0，第一个是-1，，，，[-5:-1],从右边开始数，第一个数是-1，闭合，第二个是-5就是开的。先数到的那个数是闭合的，后数到的就是开的

    #切片
    b = [1,2,3,4,5,6,7,8,9]
    b[-5:-1]
    结果：
    [5, 6, 7, 8]

1.2.5 zip函数

zip 可以将多个列表、元组或其它序列成对组合成⼀个元组列表：

    list1=[1,2,3,4]
    list2 = ["a","b","c","d"]
    list3 = zip(list1,list2)
    print(list(list3))
    结果：
    [(1, 'a'), (2, 'b'), (3, 'c'), (4, 'd')]

    print(dict(list3))
    结果：
    {1: 'a', 2: 'b', 3: 'c', 4: 'd'}

1.2.6 reversed函数

reversed 可以从后向前迭代⼀个序列：

    list(reversed(range(10)))
    结果：
    [9, 8, 7, 6, 5, 4, 3, 2, 1, 0]

1.3字典

字典就不想多说了，很多都是通用的，到用的时候在查吧。

1.4集合

集合是⽆序的不可重复的元素的集合。你可以把它当做字典，但是只有键没有值。可以⽤两种⽅式创 建集合：通过set函数或使⽤尖括号set语句：{1,2,3}

1.4.1集合取并集

合并是取两个集合中不重复的元素。可以⽤ union ⽅法，或者 | 运算符

    a = {1,2,3,4,5,6}
    b = {3,4,5,7,8,9}
    a|b
    结果：
    {1, 2, 3, 4, 5, 6, 7, 8, 9}

1.4.2集合取交集

交集的元素包含在两个集合中。可以⽤ intersection 或 & 运算符：

    a&b
    结果：
    {3, 4, 5}
